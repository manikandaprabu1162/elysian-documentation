🔧 Installation Guide

Step 1: Extract Files

Unzip the downloaded package to your local computer.

Step 2: Upload to Server

Upload all files to your web server via FTP or cPanel File Manager.

Upload destination: public_html/ or www/ folder

Step 3: Open in Browser

Navigate to your domain:

http://yourdomain.com/index.html
💡 Note: No database or server-side configuration required. This is a static HTML template.

🎨 CSS Styling Guide

All styles are in css/style.css. Key sections:

/* CSS Variables - Change these to customize colors */
:root {
    --primary: #9b7b5c;      /* Main brand color */
    --primary-dark: #7f6246;  /* Darker shade for hover */
    --secondary: #2c2a29;     /* Text color */
    --bg-light: #fefcf8;      /* Background color */
    --bg-dark: #1e1b18;       /* Footer color */
}

/* Typography */
body { font-family: 'Poppins', sans-serif; }
h1, h2, h3 { font-family: 'Playfair Display', serif; }

/* Buttons */
.btn-elegant { background: var(--primary); border-radius: 40px; }

/* Animations */
.fade-up { opacity: 0; transform: translateY(40px); transition: 0.8s; }
.fade-up.appear { opacity: 1; transform: translateY(0); }

⚡ JavaScript Guide

All JS code is in js/main.js. Includes:

  • Swiper Gallery Initialization - Responsive image carousel with autoplay
  • Scroll Animation - Fade-up effect on scroll
  • Smooth Navigation - Smooth scroll to all sections
  • Sticky Navbar - Navbar shrinks on scroll
  • Form Handlers - Contact form and newsletter submission
  • Active Nav Links - Highlights current section

Customizing Swiper Gallery:

const swiper = new Swiper('.gallery-swiper', {
    loop: true,
    slidesPerView: 1,
    autoplay: { delay: 3500 },
    breakpoints: {
        768: { slidesPerView: 2 },
        1024: { slidesPerView: 3 }
    }
});

🎨 Color Customization

Edit CSS variables in css/style.css:

:root {
    --primary: #9b7b5c;     /* Main brand color - change this */
    --secondary: #2c2a29;   /* Dark text color */
    --bg-light: #fefcf8;    /* Light background */
    --bg-dark: #1e1b18;     /* Dark footer background */
}

📌 All Sections Guide

Navigation Bar

Edit menu items in index.html:

<ul class="navbar-nav ms-auto">
    <li class="nav-item"><a class="nav-link" href="#home">Home</a></li>
    <li class="nav-item"><a class="nav-link" href="#features">Features</a></li>
    <li class="nav-item"><a class="nav-link" href="#gallery-modern">Gallery</a></li>
    <li class="nav-item"><a class="nav-link" href="#location">Location</a></li>
    <li class="nav-item"><a class="nav-link" href="#exclusive">Exclusive</a></li>
    <li class="nav-item"><a class="nav-link" href="#pricing">Pricing</a></li>
    <li class="nav-item"><a class="nav-link" href="#testimonials">Testimonials</a></li>
    <li class="nav-item"><a class="nav-link" href="#faq">FAQ</a></li>
    <li class="nav-item"><a class="nav-link" href="#contact">Contact</a></li>
</ul>

Hero Section

Edit hero content in index.html:

<section id="home" class="hero">
    <h1 class="hero-title">Where Stories Become<br>Timeless Elegance</h1>
    <p class="lead">Your description here</p>
    <a href="#gallery" class="btn-elegant">Explore</a>
</section>

Pricing Section

Each package is a column. Edit prices, features, and button links:

<div class="col-lg-4">
    <h3>€490</h3>
    <ul>
        <li>Luxury Suite</li>
        <li>Daily Breakfast</li>
    </ul>
    <a href="#contact" class="btn-elegant">Reserve Now</a>
</div>

FAQ Accordion

Add or remove FAQ items by copying the accordion-item block:

<div class="accordion-item">
    <button data-bs-target="#faq1">Your Question?</button>
    <div id="faq1">Your Answer here.</div>
</div>

Contact Form

Update form action to your mail handler:

<form action="sendmail.php" method="POST">
